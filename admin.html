Publish to Website

When you're ready, publish your changes to GitHub. Netlify will automatically rebuild and update the live site within ~30 seconds.

Shopier Hatırlatma
Rapor eklemeden önce Shopier'da ilgili ürünü oluşturduğunuzdan emin olun:
shopier.com → Ürünler → Yeni Ürün → Dijital Ürün → PDF yükle → Fiyat gir → Yayımla → Ürün sayfası URL'sini kopyala → Admin'e yapıştır
GitHub Settings
Settings → Developer settings → Personal access tokens → Fine-grained → Contents: read & write
WHAT HAPPENS WHEN YOU PUBLISH:
1. content.json is sent to your GitHub repository
2. GitHub Pages detects the change and updates the site (~30 sec)
3. aksakdunya.com shows the updated news and reports
